# Tiimo 有机极简主义

## 元信息

- **风格 ID**：`tiimo-organic-minimalism`
- **触发词**：Tiimo、有机极简、治愈系、低饱和、粉彩、ADHD 友好、纸质便利贴
- **情绪基调**：温和、安抚、无压迫感
- **参考来源**：Tiimo App（Apple Design Awards）
- **适用场景**：日程管理、专注计时、习惯追踪、心理健康辅助
- **不适用**：高频交易、硬核工具、需要强警示的合规场景

## 视觉语言

| 维度 | 规范 |
|---|---|
| 色彩 | 低饱和水彩/粉彩渐变；工作类蓝绿、休息类暖橙、过渡浅紫 |
| 饱和度 | 统一明度（Lab 空间思路），避免高刺激纯色 |
| 形态 | 柔和手绘感边缘，纸质便利贴立体感 |
| 空间 | 极细阴影与分层，大量留白 |
| 字体 | 清晰易读，避免过细字重 |

## 动效

| 类型 | 时长 | 缓动 | 说明 |
|---|---|---|---|
| 微交互 | 300ms | ease-out | 日常反馈 |
| 里程碑 | 600ms | spring | 完成庆祝（如花朵绽放） |
| 页面过渡 | 400–500ms | 模拟翻书/纸张滑动 | 比行业标准慢 0.2–0.3s |

- 减少动效降级：关闭弹簧/粒子，保留即时状态切换

## 组件模式

### 时间展示
- 优先环状时间轴，替代生硬线性进度条
- 支持「现在 / 接下来 / 之后」三段式，而非精确分钟压迫

### 反馈文案
- 避免：「未完成 ❌」
- 使用：「已调整 🔄」

### 任务完成
- 图标绽放 + 柔和音效（可选）
- 触觉：重要提醒 3 次脉冲，日常单次轻触

### 中断恢复
- 提供「一键重回」专用入口

## Design Tokens

```css
:root {
  --color-work: #5BA8A0;        /* 蓝绿：专注 */
  --color-rest: #E8A87C;        /* 暖橙：放松 */
  --color-transition: #C5B4E3;  /* 浅紫：过渡 */
  --color-bg: #F8F9FA;
  --color-text: #2D3436;
  --radius-sm: 12px;
  --radius-md: 20px;
  --shadow-soft: 0 2px 12px rgba(0, 0, 0, 0.06);
  --motion-fast: 300ms;
  --motion-slow: 600ms;
  --ease-spring: cubic-bezier(0.34, 1.56, 0.64, 1);
}
```

## 实现约束

- 推荐：Tailwind + CSS 变量；动效可用 Framer Motion
- 禁止：纯红错误、倒计时压迫感、惩罚性未完成文案
- 无障碍：对比度 ≥ 4.5:1；提供减少动效；多种输入方式（拖拽/语音/模板）

## 背景随时间渐变（概念）

```css
.background {
  background: linear-gradient(var(--current-time-hue), transparent 70%);
}
/* 早晨 210deg 浅蓝 → 中午 180deg 青绿 → 傍晚 40deg 暖橙 */
```
